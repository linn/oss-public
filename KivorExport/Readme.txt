To build the EXE:

1) Get .py script working

2) Ensure py2exe is installed (http://www.py2exe.org)

3) Execute 'python setup.py py2exe'

4) Necessary files are output in the 'dist' directory

5) Rename dist directory to KivorExport.

6) Zip up this directory including the KivorExport dirname for rls
