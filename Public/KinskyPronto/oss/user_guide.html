Kinsky Pronto User Guide

Overview

This document is the user's guide to the Kinsky Pronto DS Control Point.

  • Goals:
    This document is aimed at the end user & outlines the available functionality of the Pronto when loaded with this module.
  • None Goals:
    This document does not aim to educate the user on how to install or configure the module, that functionality is found in the developer and integrator guides.

Please note that the screenshots shown here are only examples as your installer has some flexibility as to how the UI is presented. In particular the home screen is likely to be completly different as it will have been tailered to your system, not just a single DS with two (non DS sources) as in our example.

Linn provide 3 DS activities and a further basic package to allow your integrater to control the volume and source selection of a preamp (or integrated DS). However, this latter package is not further detailed here as it is entirely custom and will depend on your installer.

General Usage

The Kinsky Pronto UI has bee designed where ever possible to be consistant in it's screen layout and key usage. Note that the UI does not use the LCD touchscreen, instead all interaction is via the physical keys on the Pronto.

On All pages the following keys operate as controls:

  • Vol +/-: control volume
  • Mute: toggle mute on/off
  • Home: return to Pronto home page
  • Soft #1: select previous track
  • Soft #2: select next track
  • Soft #3: Play/Pause current track
  • Soft #4: toggle repeat playlist mode
  • Soft #5: toggle shuffle playlist mode
  • Playlist and Library pages add
  • Scroll wheel: line up/down in list
  • Ch +/-: line up/down in list
  • Nav up/down: page up/down in list
  • Playlist page add
  • OK: play selected track
  • Menu: toggle softkeys to add
  • Soft #1: delete playlist
  • Soft #2: delete selected track
  • Soft #3: jump list view to current playing track
  • Library page add
  • OK: action selection
  • Nav left/right: navigate levels in library tree
  • Soft #1: play now
  • Soft #2: play next
  • Soft #3: play later

The Presets page allows the user to select a Internet Radio station to listen to. If there are more than one page of presets available then the Channel +/- buttons can be used to select.

Preamp Only Control

The Preamp only Control activity is designed to allow source selection and volume control for a preamp attached source which is not controlable by the DS (For example an LP-12). It is possible to have more than one such activity one per source to be selected.
