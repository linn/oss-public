Kinsky Pronto Documentation

User Guide

End user centric view of a standard Kinsky Pronto installation.

Assumed familiarity: None

Integration Guide

Intergator based view of how to use the sample .xcf files and the individual .xgf files (DS, Radio & Preamp) to easily produce custom Pronto application modules for end users.

Assumed familiarity: Pronto Developer Certification Level 1.

Developer Guide

A ProntoScript oriented guide to embeding the LinnPreamp.js (aka Basik) ProntoScript module into a custom ProntoScript module.

Assumed familiarity: Pronto Developer Certification Level 2.

Resources

The ProntoScript Installer's Guide can be downloaded from either the Philips Pronto website or remotecentral.com, as can the Developer's Guide, ProntoEdit Pro 2 software & Pronto firmware updates.
